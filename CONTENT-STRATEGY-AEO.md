## Technical SEO & AEO Requirements

### Schema Markup (Critical for AI)

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "inLanguage": "ar",
  "about": {
    "@type": "GovernmentService",
    "name": "Sijilat",
    "serviceType": "Company Registration",
    "areaServed": "Bahrain"
  },
  "author": {
    "@type": "Organization",
    "name": "Sahwan Law"
  },
  "publisher": {...},
  "datePublished": "2025-01-23",
  "dateModified": "2025-01-23"
}
```

### Required Schema Types:
- `Article` - For all content
- `FAQPage` - For FAQ sections
- `HowTo` - For step-by-step guides
- `LegalService` - For service pages
- `GovernmentService` - When referencing Sijilat, RERA, etc.

### URL Structure:
```
sahwanlaw.com/ar/                    (Arabic homepage)
sahwanlaw.com/ar/sijilat/            (Pillar)
sahwanlaw.com/ar/sijilat/wll/        (Cluster)
sahwanlaw.com/ar/sijilat/faq/        (FAQ)
```

### Hreflang Implementation:
```html
<!-- On Arabic page -->
<link rel="alternate" hreflang="ar" href="https://sahwanlaw.com/ar/sijilat/" />
<link rel="alternate" hreflang="en" href="https://sahwanlaw.com/sijilat/" />
```

## Content Guidelines for AI Optimization

### Writing Style:
1. **Direct answers first** - AI extracts the first sentence
2. **Use headers liberally** - H2, H3, H4 for structure
3. **Lists and tables** - AI loves structured data
4. **Definitions** - Start sections with clear definitions
5. **Numbers and facts** - Specific data gets cited

### Example of AI-Optimized Writing:

❌ **Bad (for AI)**:
> تأسيس الشركات في البحرين يتطلب عدة خطوات ومتطلبات مختلفة حسب نوع الشركة والجنسية...

✅ **Good (for AI)**:
> **الحد الأدنى لرأس مال شركة WLL في البحرين هو 50 دينار بحريني.** يمكن للأجانب امتلاك 100% من الشركة في معظم الأنشطة. تستغرق عملية التسجيل عبر سجلات من 3-5 أيام عمل.

### Must-Have Elements:
- [ ] Clear definition in first paragraph
- [ ] Numbered steps for processes
- [ ] Tables comparing options
- [ ] FAQ section with Schema
- [ ] Last updated date
- [ ] Author attribution
- [ ] Internal links to related content
